Data Quality Notes - Separations

Higher Null Rates

  • annualized_adjusted_basic_pay: 3.32% nulls (vs <1% in Employment)
  • grade: 1.83% nulls
  • occupational_series_code: 1.82% nulls
  • pay_plan_code: 1.91% nulls
  • work_schedule_code: 1.82% nulls
  • Mean length of service: 18.54 years (higher due to retirements)
  • Includes international duty stations: France, Japan, Mexico, India, etc.

📚 Data Quality & Privacy Notes

Privacy Redaction

  • REDACTED values appear in location fields (duty_station_*) and salary for sensitive records
  • ~48% of duty station state data is redacted in Employment files
  • Salary redaction more common in Accessions (0.57%) and Separations (3.32%)

Invalid/Missing Data Markers

  • INVALID - Data validation failed
  • * - Invalid code value
  • NO DATA REPORTED - Missing submission
  • UNSPECIFIED - Not categorized

File Naming Convention

{type}_{yyyymm}_{version}_{download_date}.csv

Example: employment_202501_1_2026-01-08.csv

Recommended Analysis Tools

  • Python: pandas, polars, dask for large file handling
  • R: data.table, arrow for efficient CSV reading
  • SQL: Load into SQLite, DuckDB, or PostgreSQL for queries
  • BI Tools: Tableau, Power BI (may need data sampling for Employment files)
